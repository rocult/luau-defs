## How to contribute to luau-defs

#### **Did you add new definitions?**

* Put all **exploit-related** functions inside [`environment.d.luau`](https://github.com/rocult/luau-defs/blob/main/types/environment.d.luau).
* **Custom definitions** (i.e. for Luraph) need their **own separate** declaration and documentation files.
* Make sure you have added the **documentation and examples** within the corrosponding [`docs file`](https://github.com/rocult/luau-defs/tree/main/docs).

#### **Did you change any definitions?**

* Update the corrosponding [`docs file`](https://github.com/rocult/luau-defs/tree/main/docs).
